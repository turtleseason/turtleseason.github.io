Jigsaw Puzzle

turtleseason.github.io/jigsaw-puzzle/

JavaScript HTML CSS React Bootstrap Reactstrap Visual Studio Code

"Jigsaw Puzzle" is exactly what it says on the tin - an in-browser jigsaw game that randomly generates puzzles from any image the browser can display.

  • Responsive layout with touch input support
  • Built with JavaScript (ES6) and React
  • Uses Bootstrap for styling and responsive layout (vanilla Bootstrap classes for most styling, and reactstrap for components requiring JavaScript)
