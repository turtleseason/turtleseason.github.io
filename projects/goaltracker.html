GoalTracker

github.com/turtleseason/goaltracker/

C# XAML WPF .NET Framework Visual Studio

A desktop application for tracking daily and weekly goals. The application colors and background image can be customized by switching between several preset themes, and new themes can be added with the built-in editor.

Made with WPF/.NET Framework, using the Model-View-ViewModel pattern for testability and easier data binding.
